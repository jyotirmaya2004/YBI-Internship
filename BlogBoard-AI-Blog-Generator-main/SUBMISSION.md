## Main Commands

Run the article generation pipeline:

```bash
python backend/run.py
```

Run in dry mode:

```bash
python backend/run.py --dry-run
```

Run AI News generation:

```bash
python backend/run.py --ainews
```

Serve the frontend:

```bash
python -m http.server 8000 --directory frontend
```
